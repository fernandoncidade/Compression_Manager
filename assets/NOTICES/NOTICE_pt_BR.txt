Aviso de Licenças e Atribuições de Software de Terceiros

Este documento lista as principais bibliotecas e componentes de terceiros utilizados no projeto COMPRESSION MANAGER e fornece referências às suas licenças. O objetivo é cumprir requisitos de distribuição (incluindo conformidade LGPL para bibliotecas Qt/PySide6) e oferecer informações para usuários e integradores. As dependências do projeto estão listadas no arquivo requirements.txt incluído no repositório/instalador.

Informações gerais sobre PySide6 (Qt para Python) — conformidade LGPL
- Biblioteca: PySide6 (Qt for Python)
- Licença: GNU Lesser General Public License (LGPL) v3 (ou conforme versão do PySide6 empacotada)
- Versão utilizada (ver requirements.txt): PySide6 6.x (conforme pacote no instalador)
- Substituição das bibliotecas LGPL:
  - As bibliotecas PySide6 e Shiboken6 são normalmente distribuídas como bibliotecas dinâmicas (DLLs) no instalador/executável.
  - Para substituir as DLLs PySide6/Shiboken6, localize a pasta de instalação/instância que contém os binários e substitua os arquivos correspondentes por versões compatíveis.
  - Código-fonte do PySide: https://github.com/qt/pyside

7‑Zip (binário externo)
- O aplicativo integra e executa o binário 7‑Zip (7zG.exe) local no sistema do usuário.
- Licenciamento do 7‑Zip:
  - 7z.dll: LGPL (a maior parte do código), com partes sob BSD 3-clause e restrição "unRAR" onde aplicável.
  - Outros arquivos: geralmente LGPL.
  - Informações e licenças: https://www.7-zip.org/ e notas de licença incluídas no pacote do 7‑Zip.
- Nota: redistribuições em binário devem acompanhar as informações de licença conforme o projeto 7‑Zip.

Declaração sobre transmissão de dados e privacidade
- O executável padrão não envia telemetria automática a servidores externos.
- Os dados do usuário (por exemplo, arquivos processados, exportações, config.json, language.json) permanecem sob controle do usuário e são armazenados localmente.
- Para detalhes sobre tratamento de dados pessoais, consulte a Política de Privacidade incluída no instalador/distribuição.
